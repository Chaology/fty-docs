# 虚拟机云桌面使用教程

泛图云 Windows 系统虚拟机自带云桌面，用户可以直接通过远程桌面软件方便操作。


## 第一步，获取虚拟机公网 IP 和账号密码

在购买或测试泛图云虚拟机后，你将收到虚拟机公网 IP 以及账户和密码。

![password](/pass.png)

## 第二步，选择合适的远程桌面控制软件

### Windows 系统

Windows 系统自带远程桌面软件。直接在任务管理中搜素“远程桌面”，打开后在 ① 输入虚拟机公网 IP，②③ 输入账户密码即可。

![win-remote](/win-remote.png)

### Mac 系统

Mac 系统并没有自带的远程桌面软件，可以免费下载微软的 [Microsoft Remote Desktop](https://install.appcenter.ms/orgs/rdmacios-k2vy/apps/microsoft-remote-desktop-for-mac/distribution_groups/all-users-of-microsoft-remote-desktop-for-mac)。Microsoft Remote Desktop 支持让 Mac 作为客户端访问运行 Windows 远程计算机。

下载安装运行，在 Microsoft Remote Desktop 主界面上点击「+」即可添加管理远程桌面连接。

![mac-remote](/mac-remote.png)

>- PC name：输入公网 IP 地址
>- User account：输入账户密码创建账户，可自动登录

![mac-connect](/mac-connect.png)

Microsoft Remote Desktop 会自动记录每一次链接的远程桌面，并保存列表，方便之后访问。
